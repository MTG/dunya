    Dunya API

    To use the api first you need to register on dunya here. Once your account is active you can start using the api, for example you can get the information of all the recordings of the makam collections with this url:
    http://dunya.compmusic.upf.edu/api/makam/recording

    In this repository you can find the code to access the API with python. For example to access the recording in the makam collection:

    from compmusic import dunya
    dunya.set_token("")
    recordings = dunya.makam.get_recordings()

    In order to get your API token you have to log in to dunya and then go to your profile where you will find your token.
