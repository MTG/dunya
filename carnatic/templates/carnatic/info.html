The CompMusic Carnatic corpus is organised in terms of concerts, recordings (songs), artists, ragas, thalas, and composers. The following table shows some numbers of this corpus:

Complementary datasets

Public datasets had been created from this corpus. These datasets are useful for various MIR related tasks on the music cultures under study in CompMusic. Please visit the respective pages for more details.

API

Dunya offers an API that allows other applications access the collections and the derived data. For more information go to deveopers page.
